Event

依縣市篩選

若有多組<select>，則需要指定id，以便在切換縣市時不會影響到其他組的郵遞區號。 

// in js
import Zipcode from './components/zipcodes.vue'
import County from './components/counties.vue'

// https://laracasts.com/discuss/channels/vue/use-a-global-event-bus
const bus = new Vue({})

...
  components: {
    TwzipcodeZipcode: Zipcode,
    TwzipcodeCounty: County
  },
  data () {
    return {
      bus
    }
  }
...
          
        

<twzipcode-county id="twzipcode__county--8"
                  selected="臺中市"></twzipcode-county>

<twzipcode-zipcode id="twzipcode__zipcode--8"
                   listen-to-county="twzipcode__county--8"
                   filter-by-county="臺中市"
                   selected="423"></twzipcode-zipcode>


<select id="twzipcode__county--8" class="twzipcode__county">
  <option value="臺北市">臺北市</option>
  <option value="基隆市">基隆市</option>
